## Native-device workflow

The private implementation went through physical-device constraints:

- Xcode signing
- Apple developer team setup
- iPhone install
- app permissions
- Meta AI registration callback
- wearable session state
- Bluetooth/wearable camera stream behavior

Those constraints are the reason the project has a public case study rather than only a demo video. The integration sequence itself is valuable engineering work.

---

## Field UX

The first native app surface exposed debug state because the integration needed visibility. Once the flow worked, the app gained a field-oriented panel so the user could operate the core loop without scrolling through diagnostics.

The lesson: build debug visibility early, then compress it into an operator-grade workflow.

## App identity

The native icon bridges the yin-yang idea of Zen with the celestial Zenith concept: balance, orbit, and a high-point star. That gives the app a recognizable identity without copying any third-party mark.
